SlopScan is a full-stack AI content forensics platform that detects whether text, audio, video, or presentations were generated by AI. The core idea is that AI-generated content leaves measurable statistical fingerprints — things like uniform sentence lengths, repetitive vocabulary, predictable language patterns — that differ from how humans naturally write and speak.
The system runs an ensemble of two approaches. First, hand-crafted statistical features — burstiness, MFCC extraction, bigram entropy — that are fast, cheap, and interpretable. Second, a fine-tuned RoBERTa transformer model trained on the mehddii/ai-text-detector-v2 dataset. The final score is a weighted combination — 40% statistical, 60% model — because ensemble methods are almost always more accurate than either approach alone.

Frontend  → React + Vite + TypeScript, deployed on Vercel
Backend   → FastAPI + Python, async REST API
Database  → PostgreSQL via Supabase
Queue     → Redis via Upstash + Celery workers
ML        → RoBERTa (text), Wav2Vec2 (audio), CLIP + ViT (video)
MLOps     → HuggingFace Transformers, custom training pipeline
